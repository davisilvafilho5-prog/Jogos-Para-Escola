# 🎮 Recursos para Escola: Jogos, ROMs, Addons e Texturas!

Bem-vindo ao projeto! O objetivo desta iniciativa é trazer novos jogos e conteúdos divertidos para o ambiente escolar. Como o projeto ainda está em andamento, novas coisas serão adicionadas ao longo do tempo (por enquanto, a seção de jogos é a mais completa).

Abaixo, você encontra o passo a passo simples de como instalar e usar cada um dos formatos disponíveis nas nossas Releases.

---

## 🕹️ Como usar as ROMs (Emuladores)

Se você baixou um arquivo de jogo para emulador (**ROM**):

1. **Baixe** a ROM desejada aqui na página.
2. **Abra** o seu emulador (como o *EmulatorJS Demo*).
3. **Arraste e solte** o arquivo da ROM diretamente dentro da tela do emulador.
4. **Escolha o console** correspondente ao jogo.
   * *Exemplo:* Se baixou o *Mario 64*, selecione a opção **Nintendo 64**.

> 🚀 **Pronto!** O jogo vai iniciar automaticamente.

---

## 💻 Como instalar os Jogos (PC)

Dependendo do formato do arquivo que você baixar, o processo muda um pouco:

### 🔹 Arquivos `.exe` (Executáveis)
* Baixe o arquivo `.exe`.
* Abra o gerenciador de arquivos do seu computador e dê **dois cliques** nele para abrir o instalador.
* Siga as instruções na tela para jogar.

### 🔹 Arquivos `.zip` (Pastas Compactadas)
* Baixe a pasta `.zip`.
* Clique com o **botão direito** nela e selecione **"Extrair Tudo"** (ou *"Extrair Aqui"*).
* Abra a pasta que foi extraída, encontre o arquivo executável (`.exe`) do jogo e dê **dois cliques** para jogar.

> 💡 **Dica:** Se o jogo que você abriu pedir alguma ROM externa para funcionar, basta baixar a ROM correspondente aqui no projeto e arrastá-la para dentro da janela do jogo aberto.

## 🟩 Como usar Texturas e Addons no Minecraft

Se você quer dar uma cara nova ou adicionar funções ao seu Minecraft:

1. **Baixe** o arquivo da Textura ou do Addon.
2. Dê **dois cliques** no arquivo baixado:
   * **Arquivos `.mcpack`** (Texturas)
   * **Arquivos `.mcaddon`** (Addons/Mods)
3. O Minecraft vai abrir sozinho e fará a importação do arquivo automaticamente.
4. Entre nas **configurações do mundo** onde você quer jogar e ative-os:
   * **Texturas:** Ative em *“Pacotes de Recursos”* (Resource Packs).
   * **Addons:** Ative em *“Pacotes de Comportamento”* (Behavior Packs).
