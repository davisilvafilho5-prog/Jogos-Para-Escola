# Jogos-Para-Escola

> **Status:bem :)** Funcionando nos PCs da escola ✅

## 📥 Download
Para baixar o jogo pronto para jogar, vá até a aba realeases
e escolha sei jogo
---

## 🛠️ Como instalar/rodar
1. Baixe o arquivo `.zip` ou `.exe` acima.
2. Extraia os arquivos para uma pasta (se for zip).
3. Clique duas vezes em `nome-do-jogo.exe` para começar.




## ⚠️ Observação
Se o Windows pedir permissão, clique em "Mais informações" e depois em "Executar assim mesmo".
